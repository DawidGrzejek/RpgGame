About The Game

This text-based RPG is built with a domain-driven design approach using C# and .NET for the backend, with Angular for the frontend UI. The game features a rich domain model with event sourcing for game state, CQRS pattern for commands and queries, and a modular architecture.

The game mechanics include character progression, inventory management, combat system, and an explorable world. Created as a demonstration of modern software architecture principles while providing an engaging gameplay experience.
